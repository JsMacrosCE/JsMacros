# JsMacros Community Edition Documentation

Welcome to the complete API documentation for JsMacros Community Edition - a Minecraft mod that allows you to write macros in various languages to automate and enhance your gameplay.

## Getting Started

Use the sidebar to navigate through:

- **Libraries**: Core global libraries available in scripts
- **Classes**: Helper classes and utilities
- **Events**: Available events and their properties

## Quick Links

- [World API](/apis/World) - Interact with the Minecraft world
- [Player API](/apis/Player) - Control the player
- [Chat API](/apis/Chat) - Send and receive chat messages
- [Hud API](/apis/Hud) - Manipulate the HUD
